# AutoBid-Testing
Practicing frontend

Practice only

A full-featured car buying and selling platform built with vanilla HTML/CSS/JS.

 Features

Authentication — Register and login with validation
Browse Listings — Search, filter by brand and car type
List Your Car — Add vehicle with all details, custom icon, price
Auction System — Place bids, live countdown timers, bid history
Direct Messaging — In-app DM between buyers and sellers
Profile Dashboard — View and manage your listings


Option 1 — Open directly in browser
Just open `index.html` in any modern browser.

 Option 2 — Run with a local server 
bash
 Using Python 
python3 -m http.server 3000

Or using Node.js
npx serve 

Then visit `http://localhost:3000`
